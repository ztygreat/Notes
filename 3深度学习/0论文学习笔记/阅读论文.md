|                      |              | (不用太注重细节、数学公式、证明等，读不懂的地方可以标记，第3遍再搞懂) | （最详细的，知道每一句话在做什么） |

| abstract（摘要）     | v            | v                                                            |                                    |
| introduction（导言） |              | v(标注重要且没读过的引用文献，如果后面仍然读不懂，可以阅读这些引用文献，或者也可以尝试上网搜索了解) |                                    |

| experiment（实验）   | 瞄一下图和表 | v(搞清楚重要的图和表)                                        |                                    |
| conclusion（结论）   | v            | v                                                            |                                    |
